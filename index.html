Enabled Arduino ECG System for Predictive Cardiac Health Monitoring

A wearable, affordable ECG device using Arduino/ESP32, capable of real-time cardiac monitoring and early warning for heart attacks via AI-based analysis.

Get in Touch

About The Project

Cardiovascular diseases are a leading global health threat. Early detection, ideally within the "golden window" (15-30 min before heart attack symptoms), can dramatically reduce risk.

This project aims to create a wearable, affordable ECG device using Arduino/ESP32, capable of real-time cardiac monitoring and early warning for heart attacks via AI-based analysis.

Our system integrates TinyML for arrhythmia detection and early prediction of myocardial infarction, with real-time alerts and wireless data transmission to mobile and cloud platforms.

Project Objectives

Develop Hardware

Create a low-cost wearable ECG system for continuous real-time heart monitoring.

Signal Processing

Acquire and filter ECG signals, extract medical features (ST-segment, QRS complex, HRV).

AI Integration

Integrate TinyML for arrhythmia detection and early prediction of myocardial infarction.

Real-time Alerts

Implement real-time alerts and wireless data transmission to mobile and cloud.

Methodology

1

ECG Signal Collection

Collect ECG signals using AD8232 sensor and ESP32/Arduino microcontroller.

2

Signal Filtering

Apply digital filtering for noise/artifact removal; motion sensor data for artifact suppression.

3

Feature Extraction

Extract relevant features (P wave, QRS complex, ST segment, HRV metrics).

4

AI Analysis

Use TinyML/CNN/XGBoost on-device for real-time prediction.

5

Alert System

Trigger alerts via buzzer, motor, LED; transmit data using Bluetooth/Wi-Fi.
